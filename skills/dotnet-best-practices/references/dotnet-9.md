# .NET 9 reference (STS)

## Positioning

An STS release. Useful for early access to platform updates and tooling improvements.
Default production landing zone remains the **latest LTS**.

## Notable themes

- SDK/tooling improvements that increase build output signal.
- Library updates and selected web/desktop improvements.

## Adopt now

- Use tooling improvements to reduce build noise and raise the quality baseline.

## Evaluate

- Adopt only when required and with a follow-on plan to land on the latest LTS.

## Avoid / caution

- Treating .NET 9 as a long-term production baseline.
